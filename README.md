# Chess App

A real-time multiplayer chess application built with React, TypeScript, and [chessground](https://github.com/lichess-org/chessground).

## Setup

```bash
git clone git@github.com:bouncychess/chess-app.git
cd chess-app
npm install
npm run dev
```
